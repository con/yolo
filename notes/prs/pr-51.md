## Summary

- Change SELinux mount label from `:Z` (private unshared) to `:z` (shared) on all runtime-managed volume mounts
- Fixes concurrent yolo instances revoking each other's file access, causing `EACCES: permission denied` crashes
- `--extra-volume` mounts left as `:Z` since users control those directly
